## 프로젝트 개요
쇼츠 콘텐츠를 자동으로 수집하고, AI로 각색한 후 이미지와 음성 합성으로 새로운 쇼츠를 생성하는 웹 플랫폼을 개발합니다.​

## 핵심 기능
- 유튜브 API를 활용해 특정 태그(예: 유머)로 조회수 높은 쇼츠를 검색하고 DB에 저장합니다.​

- 웹페이지에서 쇼츠 리스트를 확인하고, 각색 기능을 제공합니다.

- AI로 각색(후킹, 저작권 회피, 재미 요소 추가)을 수행합니다.​

- AI 이미지 생성 API를 활용해 등장인물을 생성하고, 문장별로 이미지를 삽입할 수 있습니다.​

- TTS API를 활용해 각색된 스크립트를 음성으로 합성합니다.​

- 생성된 이미지와 음성을 바탕으로 쇼츠 동영상을 자동 생성하고, 유튜브에 업로드합니다.​

- 사용자별 API 키를 등록하고 관리할 수 있습니다.

- 회원가입 및 멀티유저 지원 기능을 단계적으로 도입합니다.

## 사용자 프로세스

사용자 프로세스

1. 회원가입을 진행한다.
2. 각종 API Key를 등록한다.
    - YouTube API Key
    - LLM API Key (기본값: ChatGPT)
    - 이미지 생성 API Key (기본값: ChatGPT)
    - TTS API Key (기본값: Typecast)

3. 검색 탭에서 쇼츠 카테고리를 선택해 검색한다.  
   (예: 유머, 여행, 정치, 경제 등)  
   내부 알고리즘을 통해 ‘구독자는 적지만 조회수가 높은 쇼츠’를 우선 분류한다.

4. 사용자는 검색된 쇼츠 중 마음에 드는 영상을 선택한다.
5. 선택한 쇼츠를 기반으로 LLM API를 이용해 대본을 재작성한다.  
   (기존 영상과의 유사도는 약 85% 유지)
6. 사용자는 생성된 대본을 검토하고 수정한다.

7. 사용자는 대본에 등장할 인물을 설정한다.
8. 지정한 인물의 특징을 기반으로 이미지 생성 API를 통해 인물 이미지를 생성한다.

9. 대본을 문장 단위로 분리한 뒤, 각 문장별로 AI 이미지 생성 버튼을 제공한다.
    - 특정 등장인물을 지정하면 동일 인물로 인식되도록 시드값을 고정한다.
    - 등장인물이 지정되지 않은 경우 일반 이미지가 생성된다.

10. 각 대본 줄에 대해 Typecast API를 이용해 TTS 음성을 생성한다.

11. 생성된 이미지와 음성을 바탕으로 최종 동영상을 제작한다.
    - 예시: 3번째 줄에서 이미지 생성 버튼을 눌러 이미지를 생성했다면,  
      1~3번째 줄의 TTS 음성은 3번째 줄에서 생성한 이미지와 함께 영상에 포함된다.
    - 예시: 8번째 줄에서 이미지 생성 버튼을 눌러 이미지를 생성했다면,  
      4~8번째 줄의 TTS 음성은 8번째 줄에서 생성한 이미지와 함께 영상에 포함된다.
    - 각 이미지와 해당 구간의 TTS 음성을 결합해 쇼츠 영상을 생성한다.


## 필요한 정보
- 썸네일, 제목, 영상 태그, 영상 설명, 영상 링크 등 쇼츠 기본 정보.​ (더 추가 예정 판단해서 추가해도 가능함)

- 각색된 스크립트, 등장인물 정보, 이미지 생성 요청, TTS 요청 등 콘텐츠 생성 정보.

- 유튜브 채널 ID, 사용자별 API 키 등 사용자 정보.

## 기술 스택
- YouTube Data API (쇼츠 검색, 업로드).​

- LLM API (기본값 chatGPT 추후 다른 옵션 추가 예정)

- AI 이미지 생성 API.​ (기본값 chatGPT 추후 다른 옵션 추가 예정)

- TTS API.​ (기본값 Typecast 추후 다른 옵션 추가 예정)

- Spring/React 기반 웹 프론트엔드 및 백엔드.

- AWS 배포.

저작권 회피 및 콘텐츠 원본과의 차별화를 위한 AI 각색 전략.​

API 키 및 사용자 권한 관리 체계.​

쇼츠 비율(9:16) 및 이미지/영상 생성 최적화.​

자동화 및 배치 처리를 통한 효율적 운영.
